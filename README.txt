# Better Way Passage Website

A sleek, responsive nonprofit website designed for Better Way Passage. Built with HTML, CSS, and Font Awesome icons. Optimized for accessibility, fast loading, and mobile responsiveness.

## ✨ Features
- 🎨 Modern design with neutral palette + peach accents
- 🔒 Secure donation gateway (PayPal one-time + recurring)
- 📝 Practitioner feedback survey (Google Form embed)
- 📱 Fully responsive and accessible layout
- ⚡ Lightweight, fast-loading pages
- 🔍 Basic SEO setup for better visibility

## 📂 Pages
- **Home** – Introduction with hero banner and highlights  
- **About** – Mission, vision, and story of Better Way Passage  
- **Donate** – PayPal embed for secure donations  
- **Feedback** – Practitioner survey (Google Form embed)  

## 🛠️ Tech Stack
- HTML5 + CSS3  
- Font Awesome (icons)  
- Responsive Grid/Flexbox layout  

## 🚀 Setup & Usage
1. Download or clone this repo.  
2. Open `index.html` in your browser.  
3. Edit content/images inside the `images/` folder.  
4. Update donation link (PayPal) and survey link (Google Form) with your own.  

## 📌 Customization
- Replace placeholder text/images with your organization’s content.  
- Adjust colors and fonts in `style.css`.  
- Update SEO `<meta>` tags in the `<head>` section of each page.
